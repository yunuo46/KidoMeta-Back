## Intro
건국대학교를 배경으로 한 메타버스 가상공간 내에서 여러 사용자가 맵 내부를 이동하고 실시간으로 소통할 수 있는 서버 구축

## Stack
- Java 21
- Spring Boot
- MySQL, H2
- WebSocket with Stomp
- CI/CD : Git Actions, Docker
